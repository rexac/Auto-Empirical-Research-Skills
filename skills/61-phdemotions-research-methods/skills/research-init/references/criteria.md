## Required files and their sources

| File | Template | Purpose |
|------|----------|---------|
| `README.md` | `templates/readme.md` | Cornell template adapted for research |
| `CITATION.cff` | `templates/citation.cff` | Machine-readable citation |
| `LICENSE` | `templates/license.md` | Dual license: MIT code + CC-BY data |
| `.gitignore` | `templates/gitignore` | Research-specific ignores |
| `_targets.R` | `templates/targets.R` | R pipeline stub with educational comments |
| `R/00_setup.R` | `templates/setup.R` | R setup with research defaults |
| `Snakefile` | `templates/snakefile` | Python pipeline stub |
| `python/00_setup.py` | `templates/setup.py` | Python setup with research defaults |
| `pyproject.toml` | `templates/pyproject.toml` | Python project definition + deps |
| `reports/manuscript.qmd` | `templates/manuscript.qmd` | Quarto manuscript template |
| `reports/supplement.qmd` | `templates/supplement.qmd` | Supplementary materials template |
| `reports/_quarto.yml` | `templates/quarto.yml` | Quarto project config |
| `docs/pre-registration.md` | `templates/pre-registration.md` | Pre-reg skeleton |
| `docs/decisions/TEMPLATE.md` | `templates/decision.md` | Decision log entry template |
| `docs/pipeline-status.md` | `templates/pipeline-status.md` | Pipeline state tracker |
